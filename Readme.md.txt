This script installs Docker Deamon and launches a Docker container with a Nginx server.

1. Run the command in the console to download the script file:

     $ curl -o script.sh https://gitlab.godeltech.com/k.marozava/script/-/raw/master/script.sh?inline=false

2. Change the mod for script.sh. Run the command in the directory where the script is located:

     $ sudo chmod 500 script.sh

3. Run the script with the command:

     $ sudo ./script.sh

4. Input in command line path to your html file. Example:

     /home/user/www

Complete!
